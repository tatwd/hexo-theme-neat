# hexo-theme-neat
> A neat theme for hexo blog.

### Get Started
Run this your blog root dir
```bash
$ git clone https://github.com/tatwd/hexo-theme-neat.git themes/neat
```
Then, change your theme in `_config.yml` of root dir.
```yml
theme: neat
```
